# The Entropy-Fidelity Equilibrium Model (EFEM)

## Core Mathematical Framework
The Perceived Realism ($R$) is defined as:

$$R = f(I) \cdot e^{-\lambda | \Phi(I) - S |}$$

Where:
- $I$ is Information Fidelity
- $S$ is Stochasticity (Entropy)
- $\Phi(I)$ is the Ideal Entropy Threshold
